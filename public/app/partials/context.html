By default, twenty (20) runners are registered for this competition demo, their information (name, surname, etc.) was automatically created using fzaninotto/faker library, you can use links on navbar for getting access to:

  • Dashboard: UI, displays in real-time & table form the runners who have entered the finisth corridor.
  • Test-Client: UI where you can see a simulated behaviour of all of those twenty (20) runners during competition, TIMER starts automatically. You can see as a log and as dynamic sets the chip codes "detected" by each Timing Point during the competition.
  • API: A small set of available resources implemented on the backend which receives the timing information in real-time.
